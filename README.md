### Useful Github commands

```bash
git clone <git url>
git status
git add <file>
git commit -m "message"
git push
```
```
git pull
```
```bash
// view branches
git branch
// create branch hola
git checkout -b hola
// move to branch hola
git checkout hola
```
```bash
// delete remote branch hola
git push origin --delete hola
// delete local branch hola
git branch -d hola 					       
```
```
// merge changes from branch hola
git merge hola					     	
```
